2010-02-12

_ スーツ購入

たぶん 2 年くらい前に買ったのを最後に，しばらく 4 着のスーツを毎日順番に着ていたのだけれど，6 年くらい前に購入したスーツがそろそろダメになってきた感じなので，新しく購入することにした．というか最初はそんなつもりでもなかったんだけど，コートを買いにいったショップの馴染みの店員さんに，なんかいいのないとか聞いたら，ちょうど催事場に安くで出してるからということで，あうサイズのものをいくつか持ってきてもらった．

黒系のスーツがなかったので，うっすらと細めのストライプがはいったものを試着．まぁ，だいたい試着するものはいつもだぶつくので直すことになるのですが，今回も．スーツのラインが綺麗にでるように，そしてもともと細めのスーツをかなりよりタイトな感じに直すことに．

今日，そのお直しが終わったスーツを引取りへ．いつもの店員さんはお休みで，別の人が．どこかで見たことあるなぁと思ったら，以前，別店舗のショップで何度か会ったことのある人だったようで．お店をいくつか転々として今はここにいるんだとか．数年ぶりの再会でいろいろと話が盛り上がってしまいました．

2010-01-31

_ アバター見てきた

各所で話題にのぼっているアバターを見てきました．事前にいろいろと情報を集めた感じでは IMAX 3D 版の吹き替えで見るのがおすすめっぽいのをよく見かけたので，109 シネマズ名古屋に行くことにした．事前にオンラインでチケットが購入できるので，3 日前に座席を確保．座席は中央の少し後ろくらいでちょうどよさげ．名古屋では IMAX 3D 吹替版は 10 時 30 分からの 1 回のみ．ここのところ休みの日はいつも気がついたら昼をすぎてることが多いので，起きられるか心配だったけど，無事，起きることができた．…けど，定期忘れて取りに戻ったので，電車に乗り遅れそうになったけど．

109 シネマズ名古屋の入り口になんか行列ができててなんだろうと思ったら，「なのは」の行列だそうで．何があるのかと思ったよ…．

事前予約した場合は発券機でチケットを受け取るとのことだったけど，何故かこの日は係員が数人で手作業でチケットを渡していた．どういう順序で束になっていたのかわからないけど，自分の分が見つかるまでかなり待ったような．もしかして予約できてなかったりするのかと，ちょっとドキドキしてた．

映画の方はというと，XPAND などその他の 3D 映像は見ていないので，比較のしようもないですが，映像の奥行きが感じられてとても良かったと思います．確かに字幕版を見たとすると，視線の移動が多くなりすぎて，映像に集中しづらい部分が出てくるかも．吹き替え版でも一部字幕が右手に表示されることがあったけど，そちらに目がとられてしまう感じだったし．内容はどことなく宮崎アニメっぽい感じがしたな．最初はナヴィに違和感があったけど，最後のほうになると慣れてしまったからなのか，これはこれで受け入れられるようになってた．ハリウッド映画な展開なので，ある意味安心して見られたかも．

たぶん 2D で見ても綺麗な映像なんだろうけど，近くに IMAX 3D で見ることができる劇場があれば（といっても日本では川崎，菖蒲，名古屋，箕面の 4 館だけ），ぜひ見に行かれてはいかがでしょうか？

_ たいした手間をかけずに OS X の通信をキャプチャしてみる

Windows でパケット解析するときは Wireshark を使ってるけど，自宅の OS X の環境でとりあえずちょっと使いたいときは tcpdump が標準で利用できるので，わざわざ新たにアプリケーションを導入しなくてもなんとかなる．Mac OS X v10.5, 10.6: How to capture a packet trace にマニュアルもあったりする．

今回は全通信を解析したいわけではなくて，特定のホストとの通信の解析がしたいので，マニュアル記載のオプションとホスト名を指定するようにして解析してみた．

$ sudo tcpdump -i en0 -s 0 -B 524288 -n -X '(host www.smallstyle.com) and (port 80)'

例えばこんな具合にターミナル上で実行して，ブラウザで指定したホストにアクセスしてみたりすると，パケットの内容を見ることができた．
