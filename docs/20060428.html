_ トラックバックにサイトの縮小画像 for tDiary

idea * idea の SimpleAPI を使ったトラックバックのサイトのサムネイル表示をまねてみる．tDiary の tb-show.rb に手を加えて，ここでもトラックバックのサイトサムネイル表示をしてみました．

--- /usr/local/share/tdiary/misc/plugin/tb-show.rb.org  2006-03-18 13:07:24.000000000 +0900
+++ /usr/local/share/tdiary/misc/plugin/tb-show.rb      2006-04-24 17:28:32.533389816 +0900
@@ -175,6 +175,10 @@
                end
                r << %Q!\t\t\t\t<span class="commenttime trackbacktime">#{ comment_date( t.date ) }</span>\n!
                r << %Q!\t\t\t</div>\n!
+
+               # add simple api thumbnail
+               r << %Q!\t\t\t<span class="trackback-thumb"><a href="#{ CGI::escapeHTML( url ) }" title="#{ CGI::escapeHTML( a )}"><img src="http://img.simpleapi.net/small/#{ CGI::escapeHTML( url ) }" alt="#{ CGI::escapeHTML( a )}" width="128" height="128"></a></span>\n!
+
                r << %Q!\t\t\t<p>#{ CGI::escapeHTML( excerpt ).strip.gsub( /\n/,'<br>') }</p>\n! if excerpt
        end
        r << %Q!\t\t</div>\n!

頻繁にトラックバックをもらうこともないですが，こうしてサムネイルが表示されているとちょっとうれしい感じがしますね．

追記：画像に alt 属性などがなかったので追加．
