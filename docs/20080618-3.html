_ Firefox をインストールしたまず行う設定 (Firefox 3 版)

Firefox 3 がいよいよリリースということで，ちょっとフライング気味ですが，定番な設定なんかを紹介しようかと．Firefox 3 のインストールの参考にしてみてください．

基本はFirefox をインストールしたらまず行う設定(Firefox 2.0 版) と同じような感じです．細かいところで自分好みにしておきたいものは，user.js に定義しておくと，移行も楽チンかと思います．Firefox は Add-ons で自分好みにカスタマイズできるのが，やっぱりいいところだと思います．Firefox 3 からはおすすめの拡張とか，検索なんかもできるようになっています．

ざっくりとインストール後の設定をリストアップしてみます．

  • ツールバーを自分好みに(ブックマークツールバーをメニューバーの横に持ってきたり，ボタンを最小限にしたりとか)
  • スタートアップページをブランクに
  • ダウンロードディレクトリの設定
  • 複数のタブを閉じるときに警告しない，タブバーを常に表示する
  • JavaScript の設定(Windowのリサイズとかさせたくない)
  • フォントや言語の設定
  • クッキーの設定

目新しいものはありませんね．普通に使う分にはこれで十分かもしれません．上記の設定と重複する部分もありますが，参考までに user.js をさらしておきます．ご参考までに．

// ダウンロード完了時にダウンロードマネージャウィンドウを自動的に閉じる
user_pref("browser.download.manager.closeWhenDone", true);
// 完了したダウンロードとキャンセルされたダウンロードの履歴はアプリケーション終了時に削除
user_pref("browser.download.manager.retention", 1);

// タブを常に表示する
user_pref("browser.tabs.autoHide", false);
// バックグラウンドでタブを開く
user_pref("browser.tabs.loadDivertedInBackground", true);
// window.open() もタブで開く
user_pref("browser.link.open_newwindow.restriction", 0);

// マウスのスクロール量
user_pref("mousewheel.withnokey.numlines", 5);
user_pref("mousewheel.withnokey.sysnumlines", false);

// 点滅の無効化
user_pref("browser.blink_allowed", false);

// 検索結果を新しいタブで開く
user_pref("browser.search.openintab", true);

// 画像を自動的にリサイズしない
user_pref("browser.enable_automatic_image_resizing", false);

//右クリック禁止させない
user_pref("nglayout.events.dispatchLeftClickOnly", true);

はずせない Add-ons として 3 つあげるとすれば，Stylish，Greasemonkey，Gmail Manager といったところでしょうか．Firefox ユーザーでは定番な Add-on かと思います．

Firefox 3 のリリースを迎えるにあたって，「24 時間以内に最も多くダウンロードされたソフトウェア」という世界記録に挑戦するようです．さてさて，世界記録達成なるのでしょうか!?

_ 水平環なるものを見た

昼頃に，ふと空を見たら今まで見たことない虹が．ちょうど太陽の下に，ほぼ水平な虹のようなものが見える．よくよく見ると，少し弧を描いているみたいだ．

この現象はいったいなんだろうと，ちょっと調べてみたら，環水平アークと呼ばれるものらしく，比較的珍しい現象のよう．

一応，写真におさめてみたけど，色がうすかったのであまりきれいにとれなかったなぁ．

_ Windows 環境で Perl を exe にしようとしたらエラーになった

2 年前くらいに誰かが作った Perl を修正して，PAR で exe にしようとしたらなんかエラーが出た．

プロシージャ エントリ ポイント Perl_sv_2iv_flags がダイナミック リンク ライブラリ perl58.dll から見つかりませんでした。

よく分からないのでひとまず検索．Re: Problems with ActiveState PAR/PP -- procedure entry point error に書いてあることをやったら動いた，というのがいくつか見られたので試してみたところ，無事に動作した．

普段は coLinux や リモートの Linux マシンなんかで Perl を使うことが多いので，この環境で Windows 用の exe も作れたら楽なんだけどなぁ・・・
