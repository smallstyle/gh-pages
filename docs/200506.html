2005-06-06

_ Apacheの認証

tDiaryでの更新とかでApacheで認証しているのですが，Basic認証からDigest認証に変更しました．ブラウザも対応してるのにわざわざ暗号化されていないBasic認証を使う必要性もないということで．

で，Digest認証は，.htaccessやhttpd.confなんかに，

AuthName       "tDiary"
AuthType        Digest
Require           Valid-User
AuthDigestFile  /home/hoge/.htdigest

と，適切な部分に記述．パスワードファイルは，

$ htdigest -c .htdigest "tDiary" hoge
Adding password for hoge in realm tDiary.
New password:
Re-type new password:

で，作成する．手順としてはぜんぜん難しくないわけですが，Apacheの再起動していざ認証してみるとエラー…．Apacheのエラーログを確認してみると，

configuration error:  couldn't check user.  No user file?: /

パスワードファイルが見つからないときとかにこのエラーがでるみたいなのですが，ファイル自体のPathは間違っていません．おかしいなぁとか思っていたら，もともとBasci認証の部分を書き換えていたために，AuthDigestFileの部分がAuthUserFileになっていました．正しく書き換えると，無事，認証成功することができました．

2005-06-09

_ IE6でタブブラウザ

CNET Japan経由，Internet Explorer 6もついにタブブラウザにということで，ここのところ話題になっているIE7ですが，その機能のひとつとしてタブブラウザ機能があげられます．タブブラウザは，FirefoxやOperaをはじめIEコンポーネントのSleipnirなど多くのブラウザが搭載している機能でした．IE本体にはこれまでそうした機能はなかったのですが，今回，Microsoftが米国時間8日に公開した「MSN Search Toolbar」をインストールすることで，IE6でもタブブラウザ機能を用いることができるようになりました．

今のところ英語版しかありませんが，そのうち日本語版もリリースされるかもしれません．とりあえず，どんなもんかなとインストールしてみたのですが，雰囲気としては複数のWindowが重なって表示されているのをタブで切り替えている感じ(うまく表現できないなぁ…）で，今まで利用してきたタブブラウザとはちょっと違った感じの切替方かもしれません．ブラウザのWindow全体が切り替わる感じです．個人的にはなんかちかちかしてあんまり好きになれないなぁ，と思いました．

将来リリースされるIE7ではどのように実装されるかはわかりませんが，IE7でのタブブラウザ機能はほぼ間違いなく実現されるでしょうね．

_ Debian 辞典

先日リリースされたDebian Sargeですが，Sargeに対応したリファレンス本「Debian 辞典」を購入しました．まだ，ざっとしか目を通していませんが，すでに知っていることや，知らないことなど内容は充実していると思います．Debianユーザならとりあえず持っていたい一冊なんじゃないでしょうか．

スラッシュドットで報じられていますが，CD/DVDからインストールした場合，APTのリポジトリが安定版のstableではなくtestingになっているようです．ネットワークインストールなどの場合は，この問題は関係ないのですが，/etc/apt/sources.listの修正や，『Debian辞典』情報ページのMini Tipsに書かれている方法などで対応できます．

2005-06-29

_ 空梅雨

毎年この時期は毎日毎日雨ばかりで憂鬱になるのですが，今年はまったくもって降りませんね．今朝，明け方に久々に降雨．通勤，帰宅時や移動時に降られる雨は嫌いですが，明け方にざっと降る雨とかは気になりません．

この調子だと今年の夏は深刻な水不足になりそうな予感は誰でもすると思うのですが，このままいくと今年はかなりひどいのでは…．野菜も値上がりするだろうし，困ったもんだ．

渇水時期は琵琶湖の水位も気になるわけですが，近年では平成12年に最低水位-97cmを記録．この他に琵琶湖水位の計測が開始されて以来，平成6年の-123cm，昭和14年の-103cmが記録されているそうです．

_ Windows2000SP4向け「Rollup 1」公開

スラッシュドット ジャパン経由，Windows2000SP4向け「Rollup 1」公開の記事によると，6月28日にWindows2000 SP4向けのセキュリティロールアップパッケージが公開されたようです．つい3日前にWindows2000のセットアップを行ったところなのですが，あと1,2日まてばあんなにたくさんのパッチを落とさなくてすんでいたのか…．

_ Sleipnir2 アルファ版公開

タブブラウザのSleipnirの後継，「Sleipnir2」のアルファ版が公開されたようです．まだアルファ版のため，

このリリースは Sleipnir2 の新機能のテクノロジプレビューを目的としています。

現時点では Internet Explorer と同等の機能までしか実装しておりません。
