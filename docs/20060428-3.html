_ トラックバックにサイトの縮小画像 for tDiary

idea * idea の SimpleAPI を使ったトラックバックのサイトのサムネイル表示をまねてみる．tDiary の tb-show.rb に手を加えて，ここでもトラックバックのサイトサムネイル表示をしてみました．

--- /usr/local/share/tdiary/misc/plugin/tb-show.rb.org  2006-03-18 13:07:24.000000000 +0900
+++ /usr/local/share/tdiary/misc/plugin/tb-show.rb      2006-04-24 17:28:32.533389816 +0900
@@ -175,6 +175,10 @@
                end
                r << %Q!\t\t\t\t<span class="commenttime trackbacktime">#{ comment_date( t.date ) }</span>\n!
                r << %Q!\t\t\t</div>\n!
+
+               # add simple api thumbnail
+               r << %Q!\t\t\t<span class="trackback-thumb"><a href="#{ CGI::escapeHTML( url ) }" title="#{ CGI::escapeHTML( a )}"><img src="http://img.simpleapi.net/small/#{ CGI::escapeHTML( url ) }" alt="#{ CGI::escapeHTML( a )}" width="128" height="128"></a></span>\n!
+
                r << %Q!\t\t\t<p>#{ CGI::escapeHTML( excerpt ).strip.gsub( /\n/,'<br>') }</p>\n! if excerpt
        end
        r << %Q!\t\t</div>\n!

頻繁にトラックバックをもらうこともないですが，こうしてサムネイルが表示されているとちょっとうれしい感じがしますね．

追記：画像に alt 属性などがなかったので追加．

_ ボーリング

一年ぶりくらいだろうか，久々に会社の人たちとボーリング．急遽，決まったわりにはそれなりに集まった感じ．みんな部署もかわったりお客さん先に行ったりで，こうして会うのは久々な面々．

3 レーンに分かれてゲームをしたのですが，元々丸いものを使った競技が苦手な私は今日もスコア 100 超えるのがやっと．2 つ上の先輩と，ガータを 2 回出したり，スペア宣言して失敗したら，缶ビール 1 本あけるとか，不思議ルールで争ってみたり．結局 2 本，ゲーム中に飲みました．空きっ腹にはかなりきました…

11 人中，10 位で同率ブービー賞．後輩とじゃんけんして負けたので，賞品は後輩に譲りました．っていうか前回もブービー賞だったような．

その後ラストオーダー 10 分前に居酒屋にはいる嫌な客をして帰ってきました．んー，楽しかった．

2006-04-26

_ 健康診断

毎年，身長，体重，体脂肪の変化が気になるこの季節．今年はいったいどんな変化が…．

まず身長は 3mm 減．まぁ，これは誤差だろうな．体重は 2kg 減．もともとそんなにないのでちょっと減りすぎかも．ここ数ヶ月あんまり食べてないからなぁ．体脂肪は 2％ 減．高校生のころは 1 桁だったので，まぁ，そんなもんかな．というわけで，全体的に減少傾向．不健康気味といったところかな．最近は筋トレもぼちぼち続けているから，それなりに余分なものはそぎ落とされているのだろうが…

視力は裸眼でしっかり両目ともに 1.5 をキープ．この仕事をしていて目がいいのは珍しいんじゃないかというほど，周りはみんなメガネやコンタクト．それでも昔は 2.0 あったことを思うと，若干視力は下がったような気がする．

あと，今日は採血のときの血の出がよかった．調子が悪いとなかなか出てくれないことがあるので．ちょうど自分が採血しているときに，隣で採血されていた人は極度の注射嫌い？のようで大変そうだった．針を刺そうとすると，それを振り払おうとしてるし…．でも，苦手な人はやっぱり嫌なんだろうなぁ．

その他の結果は後日連絡ということで，問題がなければいいのですが．
