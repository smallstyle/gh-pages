_ Red Hat Linux スキルチェックテスト

2005-08-06

_ メモリ増設とkernelのアップデート

昨晩，サーバのメモリを256MBから512MBに増強しました．で，再起動ついでにkernelを2.6ベースにアップデート．woodyからそのままdist-upgradeしたsarge環境なのでブートローダーはliloを使っていたのですが，案の定というかなんというか，kernel panicで起動できず．

VFS: Unable to mount root fs on unknown-block(3,2)

といったメッセージで起動に失敗します．原因を調べるのも時間かかりそうだったので，ブートローダーをgrubに変更．

# apt-get install grub
# mkdir /boot/grub
# update-grub
# grub-install /dev/hda

これで無事起動できました．サーバの温度監視にlm-sensorsを利用しているのですが，kernelを2.6に変えたことでlm-sensorsを利用してグラフ表示しているHotSaNICの設定を変更する必要あり．HotSaNIC/modules/sensors/settingsを以下のように変更．

SENSOR=/sys/bus/i2c/devices/1-0290/temp2_input,temp2,CPU Temp,1,0.001,0,C

これでkernel2.6環境で温度の監視が可能になりました．

_ 夕立

昨日，夕方会社をでるころ遠くに黒い雲がはりだしているのを見ました．なぁんか雲行きあやしそーなんて思いながら地下鉄に乗りました．家のある方面に向かうにつれてだんだん暗くなり，空がひかり始めました．と，思ったら一気に降ってきました．夕立です．天気予報では晴れ時々くもり．傘なんて持っていません．駅についてコンビニでビニール傘を購入．駐車場までの5分ほどの距離なのですが，それでもズボンはかなりぬれてしまうほどの大雨でした．20分もすれば小雨になったのですが…．

今日も夕方から雷が鳴り出し，その後激しい雨が降りました．ほんの10分ほどの雨ですが，雨上がりは少し気温もさがりすごしやすくなります．雨は嫌いですが，夏の夕立はなんとなく嫌いになれない部分があります．雨上がり網戸にセミがとまってなきはじめた．目の前でなかれるとかなりうるさいです…．
