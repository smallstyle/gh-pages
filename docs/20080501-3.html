_ 湯谷温泉にいってきた(1)

GW を利用して温泉へ行くことに．場所は静岡と岐阜の境にある湯谷温泉へ．8 時過ぎに出発して，一路，浜松へ．東名高速は GW といいつつも，一応平日ということもあって，岡崎あたりで若干渋滞はしたものの，それほど車も多くない（っていうか，普段どれくらいか知らないけど）．三ケ日 IC で一度降りて，下道で浜松へ向かう（ ETC の通勤割引を利用したので，ちょうどここまでが 100 km 手前）．

三ケ日といえばみかん．道路沿いには，いろんなところでみかんを売っている．時期的には甘夏もおいしい時期なんだろうか．あとで買いにいこうかとか思いながら，10 時半頃に浜松市内へ．

浜松に入ると，うなぎの看板がたくさん目に入る．せっかく，浜松にきたんだし，昼ごはんはやっぱりうなぎにする？ってことで，本屋に入って，いい店がないかを探してみる．ガイドブックに掲載されている，「かんたろう」というお店を選択．店についたのが 12 時前．ちょうど昼時ということもあって，駐車場は満車．どうするかなぁ，と思ってたらちょうど 1 台出ていくところで，運よく店に入ることができた．

うなぎは関西風の焼き方で身はふっくらしつつも，表面はかりっとしていて，あっという間に食べ終わってしまった．もともとはうなぎの予定はなかったけど，思いがけずうまいうなぎにめぐりあえてよかった．機会があればまた是非食べに行こう．

おなかもいっぱいになったところで，今日の目的地である湯谷温泉を目指すことに．途中，野菜の直売所で，いちごやら甘夏を購入．なんだかんだで寄り道しながら向かったけど，14 時過ぎには目的地，湯谷温泉の泉山閣に到着．チェックインをすますと，ロビーで抹茶がいただける．部屋につくとトンビがちょうど何かもらっていたようで，たくさん飛んでいた．

部屋で一休みしたところで，時間はまだ 15 時にもなっていない．とりあえず，今日も暑かったのでまずは温泉で汗を流すことに．露天風呂でぼんやりと外を眺めながら，ゆったりとした時間を過ごす．部屋に戻って冷たい水を飲んだら，少し散歩をしてみることにした．

夕飯は，結構たっぷりな量が出てきておなかいっぱいに．どれもとてもおいしかったです．

就寝前に再び露天風呂へ．10 時過ぎということで，他の入浴者もいなくて，無料でいただくことのできる日本酒を飲みながら，のんびり時間を過ごしました．

_ VirtualBox 上の Windows XP でキーボードが反応しない

ホスト OS が Ubuntu 8.04 日本語ローカライズド版の VirtualBox 上に Windows XP をインストールしたところ，ゲスト OS 上でキーボードが反応しない．マウスは利用できるので，スクリーンキーボードを利用すれば，文字入力ができないわけではないけど，どう考えても使いにくい．

日本語環境が影響しているのだろうかと，言語設定を英語に変えてみたら，キーボードが反応するようになった．まぁ，これでも別にいいけど，なんだか気になるのでもう少し調べてみると，Ubuntu Forums で同様の現象についてのスレッドがあった．

ざっと見てみると，言語サポートの「多言語入力のサポートを有効にする」のチェックを外すといいとあったので，試してみたけど，これだとホスト OS 上で，日本語入力ができなくなってしまう．もう少し読んで見たところ，scim-bridge-client-qt をインストールすると解決できるとある．

sudo apt-get install scim-bridge-client-qt

で，インストールして reboot して確認してみると，ゲスト OS の Windows XP 上でもキーボードが利用できるようになった．

_ 待ちうけカレンダーメーカー for 705NK(6) - 振替休日に対応しました

カレンダーの休日の扱いが違うとの指摘があったので待ちうけカレンダーメーカーを修正しました．振替休日になる基準が祝日法改正によって変更になっていたようです．Wikipedia の振替休日によると，

2005年の祝日法改正（2007年施行）により、祝日が日曜日である場合は、その直後の「国民の祝日でない日」を休日とする事になった（振替先が月曜日固定では無くなった）。その月曜日以外の振替休日初の適応予定日は2008年の5月6日（火）とされている。同年のみどりの日（5月4日）が日曜日となり、翌5日（月）はこどもの日でやはり祝日であることから、みどりの日の振替分が6日の火曜日となった。

とのこと．ついこの前まで振替休日になるなんて知らなかったわけで，あやうく出勤するところでした．

まだ 4 月なので，5 月分のカレンダーは 2 ヶ月表示で利用されている方が主だと思いますが，正しいカレンダーが表示されるようになっているのでご確認ください．
