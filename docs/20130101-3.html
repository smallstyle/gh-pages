2012-12-31

_ 2012 年振り返り

日記を読み返してみると，ここ数年と変わらずあまり日記という体裁にはなっていないけれども，少し振り返ってみる．

年初には何か新しいものを作って公開したいなという目標をたてていたけど，2 月に NFC を利用して Foursquare にチェックインする Android アプリ「FastTagCheckin」を新しくリリースできた．アプリは海外でもちょこっと利用してもらっているみたい．今年は国内でも NFC 対応の端末がもっと増えてくるかと思ったけど，それほどでもなかったような感じ．来年はどんな動きになっていくかなぁ．来年も NFC 関連で Android アプリを何かアイデアが生まれたら作りたい．

tDiary 関連では Picasa プラグインの改良を少しと絵日記プラグインの改良ができた．何か新しいプラグインも作れたらとは思っていたけど，今年は何も作れなかったなぁ．来年は何か出したいところ．

健康面では虫歯が進行してしまって治療が必要になってしまったことは残念．治療後も極端に冷たいものや熱いものはまだちょっとしみたりするけど…．

個人的には 2012 年後半はぱっとしない感じもしているので，2013 年は前半から継続して活動していきたいな．

2012-12-10

_ Ubuntu 12.10 な KVM ゲストに KVM ホストから仮想シリアルコンソールでアクセスする

KVM を利用した仮想環境を作っていて，KVM ホストは固定 IP を割り当てていて，KVM ゲストはブリッジ接続で，DHCP サーバーから IP アドレスを割り当てるような環境になっている．ゲストを再起動したときとかに，割り当てられた IP アドレスが変わってしまうこともあるので，ホストから簡単にアクセスできる仮想シリアルコンソールを利用できるようにした．

はじめに，ゲストにコンソールデバイスが定義されているか，以下のコマンドを実行して確認する．

# virsh ttyconsole my_vm

既に定義されていれば，例えば /dev/pts/1 などが表示される．

次に，ゲストにシリアルコンソールの設定を追加する．以下のようなファイルを /etc/init/ttyS0.conf として作成する．

# ttyS0 - getty
#
# This service maintains a getty on ttyS0 from the point the system is
# started until it is shut down again.

start on stopped rc RUNLEVEL=[2345]
stop on runlevel [!2345]

respawn
exec /sbin/getty -L 115200 ttyS0 xterm

その後，以下のコマンドを実行してシリアルコンソールを初期化する．

$ sudo start ttyS0

ホストから以下のコマンドを実行すると，login プロンプトが表示されてアクセスが可能になった．

# virsh console my_vm
Connected to domain my_vm
Escape character is ^]

Ubuntu 12.10 ubuntu ttyS0

ubuntu login:

参考：Direct terminal access via Serial Console
