_ コメントフォーム非表示プラグイン

tDiary では コメント spam 防止のために一定期間を過ぎた日記に対してのコメントは破棄する設定が可能です．ちょっと前に以前の日記でもコメントができるように設定変更しましたが，コメント非表示プラグインを作ってみたので，再度 1 年以上前の日記へのコメントは破棄する設定に戻しました．

#
# comment-hide plugin
#
if @mode == 'day' or @mode == 'month' and not bot? then
   if @conf.options.include?('spamfilter.date_limit') &&
         @conf.options['spamfilter.date_limit'] &&
         /\A\d+\z/ =~ @conf.options['spamfilter.date_limit'].to_s
      @date_limit = @conf.options['spamfilter.date_limit'].to_s.to_i
   else
      @date_limit = nil
   end

   if @date_limit
      now = Time.now
      today = Time.local(now.year, now.month, now.day)
      limit = today - 24*60*60*@date_limit
      if @date < limit
         add_header_proc do
            <<-STYLE
   <style><!--
      div.body+div.comment>div.caption,div.form{display:none !important;}
   --></style>
            STYLE
         end
      end
   end
end

非表示とはいうものの CSS で非表示にしているだけで，フォーム自体は残っています．IE や Firefox などのブラウザでアクセスされた場合で，CSS が有効な場合には表示されませんが，w3m などのテキストブラウザや CSS を切った状態で見るとフォームは表示されます．実際にコメントをもらっても受け付けられないのですが…

コードの大部分は tDiary の spam.rb から流用したものなので，動きそのものは問題ないはず．

_ ESCキーでアニメーション GIF を停止させる

画像読み込み中なんかで，Ajax loading gif のようなアニメーションが利用されたりするのはいいと思うし，読み込み中だとユーザーに知らせるのにも効果的だと思う．ここでも，lightbox JS を使った画像表示では，読み込み中のアニメーションを使ってるし．

でも，blog のサイドバーなんかで，新着を目立たせるように，ちかちかと点滅するようなアイコンを設置しているところがたまにあるのですが，これが読み手にとってはかなり邪魔…というか気が散る．本文を読んでる横でちかちかされても困るよなぁ．

ということで，なんとかならないかなと調べてみたら，どうも を押すとアニメーションを中止してくれるようです．また，about:config で image.animation_mode を once に設定すると 1 回だけアニメーション表示をし，none に設定するとアニメーション効果を無効にすることが可能になります．

loading gif アニメーションなんかは 1 回だけで止まってしまってもなんだかさみしいので， を押してアニメーションがとめられるというのはなかなか使えそうです．

視力は裸眼でしっかり両目ともに 1.5 をキープ．この仕事をしていて目がいいのは珍しいんじゃないかというほど，周りはみんなメガネやコンタクト．それでも昔は 2.0 あったことを思うと，若干視力は下がったような気がする．

あと，今日は採血のときの血の出がよかった．調子が悪いとなかなか出てくれないことがあるので．ちょうど自分が採血しているときに，隣で採血されていた人は極度の注射嫌い？のようで大変そうだった．針を刺そうとすると，それを振り払おうとしてるし…．でも，苦手な人はやっぱり嫌なんだろうなぁ．

その他の結果は後日連絡ということで，問題がなければいいのですが．

@@ -175,6 +175,10 @@
                end
                r << %Q!\t\t\t\t<span class="commenttime trackbacktime">#{ comment_date( t.date ) }</span>\n!
                r << %Q!\t\t\t</div>\n!

+               r << %Q!\t\t\t<span class="trackback-thumb"><a href="#{ CGI::escapeHTML( url ) }" title="#{ CGI::escapeHTML( a )}"><img src="http://img.simpleapi.net/small/#{ CGI::escapeHTML( url ) }" alt="#{ CGI::escapeHTML( a )}" width="128" height="128"></a></span>\n!
