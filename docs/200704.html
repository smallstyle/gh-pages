2007-04-17

_ SSH で外部からアクセスする

外部からアクセスできるように SSH をインストール．

$ sudo apt-get install ssh

Poderosa の SSH 鍵生成ウィザードで秘密鍵と公開鍵を生成して，作成した公開鍵を，~/.ssh/authorizedkeys に登録する．/etc/ssh/sshd_config で以下の項目を設定．

PermitEmptyPasswords no
パスワード認証は不可
PermitRootLogin no
root でのログインは不可

これで SSH を再起動して，接続できるか確認してみる．

2007-04-19

_ サーバの時刻あわせ

サーバの時刻あわせのために ntpdate をインストール

$ sudo apt-get install ntpdate
$ sudo ntpdate xxx.xxx.xxx.xxx
22 Apr 21:54:19 ntpdate[18806]: adjust time server xxx.xxx.xxx.xxx offset 0.229080 sec

定期的に時刻あわせをするために，cron に登録しておく．実行スクリプトは次のものを利用．

#!/bin/sh
/usr/sbin/ntpdate -s xxx.xxx.xxx.xxx

こいつを /etc/cron.hourly あたりに保管する．これで 1 時間ごとに時刻あわせが実行される．ちなみに -s オプションは syslog に実行ログを出力する設定．

_ 更新した日にバナーを表示するプラグイン

Updated Todayプラグインの色違いバナーを勝手に作ってみた｜caramel*vanillaを見て，tDiary でも同じようなことをやってみようということで作ってみた．

if @mode == 'day'
   add_body_leave_proc do |date|
      today = Date.today
      diary = @diaries[date.strftime( '%Y%m%d' )]
      lm = Date.parse( diary.last_modified.to_s )

      if today == lm
         <<-EOF
<style type="text/css" media="screen">
#updated_today{
position: absolute;
display: block;
top: 0px;
left: 0px;
height: 120px;
width: 120px;
z-index: 99;
}
</style>
<img id="updated_today" src="img/updated.png" alt="updated_today">
         EOF
      end
   end
end

Updated Todayプラグインの色違いバナーを勝手に作ってみた｜caramel*vanillaで配布されているバナーでお好きな画像を img/ ディレクトリ以下に　updated.png として配置すると画像のような感じに表示されるはず．

日記の最終更新日と現在の日付を比較しているので，古い日記を更新したときでもその日のうちはこのバナーが表示されます．23 時に更新すると，日付が変わった 0 時にはバナーが消えてしまいますが…．ま，Updated Today なのでいいだろう．

_ gem が使えなくなった

gem でインストールしようとコマンドをたたいたら以下のエラーが出て使えなくなってた．

D:\>gem install hoge
ERROR:  While executing gem ... (NoMethodError)
    undefined method `refresh' for #<Hash:0x2e97f08>

ぐぐってみるとすぐに解決策が．source_cache を消してやるとうまいこといくらしい．Windows にインストールしたマシンでは D:\tools\ruby\lib\ruby\gems\1.8　以下に source_cache があったので消してやったらうまくいった．
