_ MySQL->DB2

MySQLでのLIMITはDB2では使用できなかった．で，実際はどういう風に書けば同様の結果が得られるのか．

MySQL> SELECT * FROM hoge=tbl LIMIT 1,10;
DB2>   SELECT * FROM hoge_tbl FETCH FIRST 10 ROWS ONLY

みたいに書くと同様の結果が得られた．
