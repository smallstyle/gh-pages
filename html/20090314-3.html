2009-03-14

_ 日間賀島にふぐを食べに行ってきた

昨年も訪れた日間賀島．今年もふぐを食べに行ってきました．昨年同様，名鉄の日間賀島　ふぐづくしプラン．前回はデラックスプランにして，たこやら伊勢エビやらがついてたけど，今回はスタンダードプランで．

前日はかなり天候が悪く，今日も朝は雨がぱらつき，強い風が．とはいうものの，島への船が出せないとか，そんな状況でもなかったですが．今日は土曜ということもあって，定期便の船に加えて，臨時便も出ていた．

島についてから目的地に向かう中，強い雨風に傘を壊されそうになりながらもなんとか到着．あとで知ったことだけど，港から連絡すれば車で迎えにきてもらえたんだな．

今年も，てっさにはじまり，唐揚げやてっぴ，てっちりなどなど，プラン通りのふぐづくしです．おばちゃんのすすめで，今回ははじめてひれ酒を飲んでみたけど，飲みやすくておいしかった．おまけ？で，白子入りの茶碗蒸しもいただけて，これもまたかなりおいしかった．

12 時前から食べ始めて，食べ終わったのが 14 時前．思ったより長い時間かけて食べてたんだなぁ．

午後からは風が強いものの，天気は若干回復してきたので，少しばかり散歩に．案内図に緑地公園とかあったので，ぷらぷらと見に行ってみたら…公園？なかなか不思議な光景でした．

帰りの海は風が強くかなり荒れた天候に．なかなか激しい揺れの中，帰途についたのでした．

_ 認証が必要な Proxy 環境で gem を使う

Windows 上で認証が必要な Proxy 環境で gem を利用するときは，以下のおまじないを実行してから利用する．

C:\> set http_proxy=http://<username>:<password>@<hostname>:<port>

_ sqlite3-ruby のインストールが失敗する

何も考えずに，

C:\> gem install sqlite3-ruby

とするとエラーになる．なので古いバージョンの 1.2.3 を利用する．

C:\> gem install sqlite3-ruby -v 1.2.3

_ Redmine と VisualSVN

VisualSVN をいれて Redmine と連携させる．普通にリポジトリを指定すると，エラーになってしまったんだが，Path に VisualSVN の bin を通しておくと参照できるようになった．
